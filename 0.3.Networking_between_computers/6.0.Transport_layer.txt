-> Transport layer is the most important layer which directly provides services to application layer.
-> Transport layer is implemented inside OS kernel
-> Segmentation , error control and flow control takes place at transport layer
-> Transport layer is responsible for delivering packet to proper PROCESS within a machine.


Transport layer has two major protocols:-
1. TCP(Transmission control protocol) 
2. UDP(User datagram protocol)
3. SCTP(Stream control transmission protocol)
